#《Ceph设计原理与实现》勘误

首先，感谢大家对《Ceph设计原理与实现》这本书的支持。

由于各种不可控原因，书中出现了一些需要修正的信息，在各位读者的共同努力下，我们在不断进行此类信息的收集，以期能在下一次印刷中进行修正。

如果您在阅读本书的过程中，有一些疑问或者发现了一些错误的信息，欢迎添加微信scienceluo，加入作者在线答疑群进行反馈和交流。

目前收集和发现的勘误信息如下：

Page 8:

type:rook 应该为 type:root

weiqht 应该为 weight

Page 53:

倒数第三行 256KB 应该为 32K

Page 135:

流程图中 克隆对象修复下方的 是 应该为 否

Page 150:

两个NotBackfilling状态，其中左下方的 NotBackfilling 应该为 Backfilling

Page 274:

图中下面的 Libvirt 应该为 QEMU

Page 277:

第10行的命令中 ceph 应该为 DEFAULT

Page 282:

第20行的命令中 ceph 应该为 DEFAULT
